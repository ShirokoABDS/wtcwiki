Wtcraft 服务器玩家插件使用教程

本页汇总了服务器全部玩家可用插件的核心指令与使用说明，按功能模块分类整理。如仍有疑问，可在玩家群内询问，我们会持续完善本教程。

登录插件 账号安全

用于离线模式下的账号注册、登录、改密与邮箱找回。首次进服必须注册，之后每次上线需登录。

注册 / 登录

指令别名说明
/register <密码> <重复密码>/reg首次进服注册账号，两次密码需一致
/login <密码>/l每次上线登录账号

修改密码

指令别名说明
/changepassword <旧密码> <新密码> <重复新密码>/changepw修改登录密码，需提供旧密码

绑定邮箱（强烈建议）

绑定邮箱后，如忘记密码可直接通过邮箱接收验证码找回，无需联系管理员。

  1. 执行 /bindemail set 你的邮箱@example.com（或 /bdmail set），系统会向该邮箱发送验证码。
  2. 收到验证码后执行 /bindemail verify 验证码（或 /bdmail verify）完成绑定。

忘记密码（已绑定邮箱）

  1. 执行 /resetpassword forget（或 /repw forget），系统会再次向绑定邮箱发送验证码。
  2. 执行 /bindemail re 验证码 新密码（或 /bdmail re 验证码 新密码）设置新密码。
提示：密码请勿使用与其它平台相同的密码；建议进服后第一时间绑定邮箱，以免遗忘密码后无法找回。

皮肤插件 外观

支持直链皮肤、正版账号自动皮肤、LittleSkin 皮肤三种方式。

指令说明
/skin url "链接"使用皮肤直链（.png）设置皮肤，链接需用英文双引号包裹
/skin update更新当前皮肤（正版玩家换皮肤后同步）
/skin clear清除当前皮肤，恢复默认

三种获取皮肤方式

  1. 直链方式：已有皮肤 .png 直链时，使用 /skin url "https://example.com/skin.png" 即可。
  2. 正版账号：使用正版账号加入服务器，正常情况下服务器会自动为您应用对应正版账号的皮肤。
  3. LittleSkin 方式：前往 LittleSkin Getter 获取 littleskin 皮肤，或前往 SkinsRestorer Upload 上传，将网站给出的指令直接粘贴到服务器执行即可。
提示：直链需为可公开访问的 .png 图片地址（推荐 64×32 或 64×64 格式），切勿使用需要登录才能查看的链接。

举报玩家 秩序

遇到违规行为可快速举报，举报理由通过 CUI 界面选择。

指令别名说明
/report <玩家名>/rp弹出 CUI 界面，选择举报理由后提交
提示：举报请如实填写，恶意举报可能被处理。

领地插件 Dominion 保护

用于圈地保护建筑与箱子，防止他人破坏、偷窃。Dominion 是支持 Folia 的全新领地系统，可平替纯净版 Residence。

领地功能较为丰富（圈地、权限、子领地、传送点等），完整教程请访问官方 Wiki：

打开 Dominion 玩家文档

注意：建议在建筑周围圈好领地后再放置贵重物品；未圈领地的区域无法得到保护。

经济插件 交易

查看余额、玩家间转账、官方商店与玩家市场。

指令说明
/bal查看自己的余额
/eco <玩家名>转账给指定玩家
/warp WTC官方商店传送到官方商店（购买基础物资）
/market打开玩家自主交易市场

玩家传送 移动

向他人发起传送请求，需对方同意；也可拉黑玩家避免被打扰。

指令说明
/tpa <玩家名称>向该玩家发送「传送到他身边」的请求
/tphere <玩家名称>请求该玩家传送到你身边
/tpaccept接受传送请求（也可点击聊天框 [接受] 直接接受）
/tpdeny拒绝传送请求（也可点击聊天框 [拒绝] / [拒绝并拉黑]）
/denys add <玩家名称>拉黑指定玩家（不再收到其传送请求）
/denys remove <玩家名称>解除拉黑
提示：被频繁打扰时，可直接点击聊天框中的「拒绝并拉黑」一键处理。

传送点 移动

服务器预设的公共传送点。

指令说明
/warp <传送点>传送到指定公共传送点（如 WTC官方商店）

家 移动

玩家自定义的私人传送点，可设置多个并指定默认家。

指令说明
/sethome <家名称>在当前位置设置一个家
/home <家名称>传送到指定家
/homes列出已设置的全部家
/setdefaulthome <家名称>将某个家设为默认（无参 /home 时传送目标）
/delhome <家名称>删除指定家

其他命令 工具

指令说明
/back传送到上一次的位置（如死亡点、传送前的位置）
/rtp随机传送到世界某处
/tpa setlang <clear/语言>设置客户端显示语言；clear 清除自定义恢复默认

GSit 坐姿插件 趣味

在方块上坐下、躺下、转圈、趴卧、爬行。也可直接右键楼梯/台阶/地毯坐下。

指令别名说明
/sit/gsit在方块上坐下
/lay/glay在方块上躺下
/bellyflop/gbellyflop趴在方块上（肚皮贴地趴卧）
/spin/gspin在方块上原地转圈
/crawl/gcrawl在地面爬行（可通过 1 格高缝隙）
提示：姿势与爬行功能仅适用于 Minecraft 1.17+ 版本。

GMusic 音乐插件 趣味

在游戏内点播音乐，支持搜索与播放列表。

指令说明
/music打开音乐菜单，选择并播放音乐

LiteSignIn 签到 福利

每日签到获取奖励，签到界面可查看连续签到天数与奖励列表。

指令说明
/litesignin gui打开签到菜单，进行每日签到

BlockLocker 箱子锁 保护

通过在箱子上放置告示牌自动上锁，无需潜行。

  1. 手持告示牌，对着箱子（或其他容器）右键放置即可自动锁定，告示牌会显示你的ID。
  2. 只有告示牌上的玩家才能打开该容器。
重要安全提示：即便使用了告示牌箱子锁，仍强烈建议将箱子搬到家中并圈好领地。熊孩子会使用漏斗从底部吸走箱子中的物品；不圈领地还会导致基地被炸。箱子锁无法防漏斗，领地才是根本保护。

G键菜单 快捷

一个 GUI 菜单，代替了旧版的 /cd 指令，集中常用功能入口。

用法：按下键盘上的 G 键即可打开菜单（如未生效，请检查客户端按键冲突）。

常见问题

Q1：忘记密码且未绑定邮箱怎么办？

A：请在玩家群内联系管理组，提供账号信息进行人工核验后重置。后续请尽快执行 /bindemail set 绑定邮箱。

Q2：皮肤设置后别人看不到？

A：让对方重新进入视野或执行 /skin update 刷新；若使用直链，确认链接为 .png 且可公开访问。

Q3：箱子锁了还是被偷？

A：箱子锁无法防止漏斗吸取。请将箱子搬到领地内，并圈好领地关闭容器权限。

Q4：传送请求没反应？

A：可能被对方拉黑（/denys）；或对方未注意聊天框提示，可提醒其点击 [接受]。

Q5：领地怎么圈？

A：详见 Dominion 官方玩家文档。
